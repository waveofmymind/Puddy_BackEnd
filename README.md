# 퍼디는?

퍼디(Puddy)는 'Pet'과 'Buddy'의 합성어로,

반려견을 가족과 친구처럼 소중하게 생각하는 의미를 담고 있습니다.

반려견의 주인이 서로 정보를 교환하고, 경험을 공유하며 도움을 주고받을 수 있는 플랫폼입니다. 

또한, 인증된 전문가들을 통해 궁금증을 해결할 수 있습니다.

## 기술 스택

### BackEnd

- Java 17
- Spring Boot 2.7.9
- Spring Security
- Gradle
- JPA
- Querydsl
- S3
- AWS RDBMS
- MySQL
- Redis
- GCP Ubuntu 20.04
- Docker
- Jenkins
