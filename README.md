## 기술 스택

### BackEnd

- Java 17
- Spring Boot 2.7.9
- Spring Security
- Gradle
- JPA
- Querydsl
- S3
- AWS RDBMS
- MySQL
- Redis
- GCP Ubuntu 20.04
- Docker
- Jenkins
